README - Place your Service Overview CSV files here

The analysis tool will automatically use the most recently modified CSV file from this directory.

Expected columns in Service Overview files:
- AssetID (phone numbers)
- Username (user assignments)
- Service types and status information

Example filenames:
- Service_Overview_20251213.csv
- ServiceOverview_Export_Dec13.csv
- latest_service_data.csv

The tool supports any CSV filename - it will find the newest file automatically.
