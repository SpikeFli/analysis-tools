README - Place your User Management CSV files here

The analysis tool will automatically use the most recently modified CSV file from this directory.

Expected columns in User Management files:
- User identification information
- Account status and permissions
- Contact information
- Role and access level data

Example filenames:
- UserManagement_20251213.csv
- UserMgmt_Export_Dec13.csv
- user_management_latest.csv

The tool supports any CSV filename - it will find the newest file automatically.
